Supported Css Properties

To set colors on tag by using class, here is the name rule:

cc-[ {} | bg- | border- | border-top- | border-right- | border-left- | border-bottom- | text-shadow- | box-shadow- | outline- ][ {} | hover- | active- | visited- | focus- ]color-[ {color_name} ]

The class name should start with "cc-" and end with "color-[color name]"

There are 2 options can be used, first is the type and the second is the pseudo element.

Ex: cc-bg-color-white

Ex: cc-outline-hover-color-red

Adjustion Class Names

[ hue | saturation | lightness | alpha ][ {} | -i | -d ][ {numbers} ][ {} | p ]

First section is the type to be changed.

Second section is how the value changes, leave black to set to the given value, "-i" means to increase, "-d" means to decrease the value.

The tailed p means to change value by percentage of current value.

This type of classes require a color class.

Ex: cc-color-cc_text lightness-i50p

Ex: cc-border-color-cc_border saturation100

Setting Colors

Setting colors to use is simple, just create a style tag with "cc_config" id and use the rule:

.cc-set-[ {color_name} ] { color: {color_code}; }

Then the color can be apllied by the class rule.

You can use javascript to change colors too.

Ex: .cc-set-primary { color: #f00; }

Ex: .cc-set-whatever { color: #0f0; }

Using Javascript

There are some provided method:

Color.CSS.setColors( { color_name_1: color_code_1, color_name_2: color_code_2, ... } )

This method overwrite colors and update all css rules.

Ex: Color.CSS.setColors( { primary: "#00f" } )

Color.CSS.getColors( regex or array of regexes )

This method returns matched colors. Regexes will be leaded by ^ and be tailed by $.

Ex: Color.CSS.getColors( [ "cc_.*?" ] )

Color.CSS.removeColors( regex or array of regexes )

This method removes matched colors and update color rules.

Ex: Color.CSS.removeColors( [ "secondary" ] )

Color.CSS.setTextShadow( { x: x, y: y, size: size } )

This method changes the position of cc-text-shadow-color-[color_name] style.

Default is { x: 0, y: 0, size: "0.3em" }, not taking effect to previous style except update.

Ex: Color.CSS.setTextShadow( { x: "2px", y: "2px", size: "6px" } )

Color.CSS.setBoxShadow( { x: x, y: y, size: size } )

This method changes the position of cc-box-shadow-color-[color_name] style.

Default is { x: 0, y: 0, size: "0.3em" }, not taking effect to previous style except update.

Ex: Color.CSS.setBoxShadow( { x: "2px", y: "2px", size: "6px" } )

Color.CSS.updateRules()

This method update color rules.

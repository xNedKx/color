Supported Css Properties


To set colors on tag by using class, here is the name rule:




cc-
[ <css_type>- or null ]
[ <pseudo_type>- or null ]
color-
[ color_name ]




The class name should start with "cc-" and end with "color-[ color name ]"


The css types are: 


bg-
border-
border-top-
border-right-
border-left-
border-bottom-
text-shadow-
box-shadow-
outline-


The pseudo types are: 


hover-
active-
visited-
focus-




Ex:


cc-bg-color-orange


cc-border-bottom-hover-color-red








Adjustion Class Names




[ <css_type>- or null ]
[ <pseudo_type>- or null ]
[ adjustion type ]
[ <option> or null ]
[ numbers ]
[ <p_option> or null ]




The adjustion types are:


hue
saturation
lightness
alpha


The options are:


-i
-d


"-i" means to increase the type by number, "-d" means to decrease.


P option:


p


The p option has only one possibile value "p" for using the number as percentage.


This type of classes require a color class.




Ex:


cc-bg-color-yellow bg-hue-i30


cc-outline-hover-color-blue 
outline-hover-saturation-d50


cc-color-green lightness-i50p


cc-border-color-purple alpha50








Other Classes




[ <css_type>- or null ]
[ <pseudo_type>- or null ]
[ "contrast" or "overwrite" ]




"contrast" sets color to contrast color of the adjusted color.


"overwrite" append "!important" parameter to the css and overwrite other color styles.




Ex:


cc-bg-color-blue bg-saturation70 
bg-lightness40 cc-color-blue 
saturation70 lightness40 
contrast overwrite








Setting Colors


Setting colors to use is simple, just create a style tag with "cc_config" id and use the rule:




.cc-set-[ color_name ]
{ color: [ color_code ]; }




Then the color can be apllied by the class rule.


You can use javascript to change colors too.




Ex:


<style id="cc_config">
.cc-set-primary { color: #f00; }
.cc-set-whatever { color: #0f0; }
</style>








Using Javascript


There are some provided method:




Color.CSS.setColors( { 
color_name_1: color_code_1, 
color_name_2: color_code_2, 
... } )


This method overwrite colors and update all css rules.




Ex:


Color.CSS.setColors( { primary: "#00f" } );








Color.CSS.getColors( regex or 
array of regexes )


This method returns matched colors. Regexes will be leaded by ^ and be tailed by $.




Ex:


Color.CSS.getColors( [ "cc_.*?" ] );








Color.CSS.removeColors( regex or 
array of regexes )


This method removes matched colors and update color rules.




Ex:


Color.CSS.removeColors( [ "secondary" ] );








Color.CSS.setBorder( { 
style: border_style, 
width: border_width } )


This method changes the parameter applied to borders.


Default is { style: "solid", width: "1px" }, not taking effect to previous style except update.


Setting width to 0 or style to "none" will prevent these 2 properties.




Ex:


Color.CSS.setBorder( { style: "dashed", width: "2px" } );








Color.CSS.setOutline( { 
style: outline_style, 
width: outline_width } )


This method changes the parameter applied to outlines.


Default is { style: "solid", width: "1px" }, not taking effect to previous style except update.


Setting width to 0 or style to "none" will prevent these 2 properties.




Ex:


Color.CSS.setOutline( { style: "none", width: 0 } );








Color.CSS.setTextShadow( { 
x: x, y: y, 
size: size } )


This method changes the position of cc-text-shadow-color-[color_name] style.


Default is { x: 0, y: 0, size: "0.5em" }, not taking effect to previous style except update.




Ex:


Color.CSS.setTextShadow( { x: "2px", y: "2px", size: "6px" } );








Color.CSS.setBoxShadow( { 
x: x, y: y, 
size: size } )


This method changes the position of cc-box-shadow-color-[color_name] style.


Default is { x: 0, y: 0, size: "0.5em" }, not taking effect to previous style except update.




Ex:


Color.CSS.setBoxShadow( { x: "2px", y: "2px", size: "6px" } );








Color.CSS.updateRules()


This method update color rules.




Ex:


Color.CSS.updateRules();
